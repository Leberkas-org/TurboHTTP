## 4.5.  HTTP Upgrade

## 4.5  HTTP Upgrade

   HTTP/3 does not support the HTTP Upgrade mechanism (Section 7.8 of
   [HTTP]) or the 101 (Switching Protocols) informational status code
   (Section 15.2.2 of [HTTP]).
