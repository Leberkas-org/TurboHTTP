## Implementation Notes

- **Encoder:** `Http10Encoder.Encode()` serialises `HttpRequestMessage` to bytes using `ref Span<byte>` for zero-allocation patterns.
- **Decoder:** `Http10Decoder` is stateful, maintaining `_remainder` for incomplete messages across TCP boundaries. Implements `TryDecode()` / `TryDecodeEof()` / `Reset()`.
- **No persistent connections:** HTTP/1.0 defaults to `Connection: close`. Keep-alive is opt-in via `Connection: keep-alive` header.
- **No chunked encoding:** Content-Length is required for request bodies.
